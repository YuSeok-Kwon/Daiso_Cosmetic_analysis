## 2. Aspect 정의

## 3. 핵심 구분 기준

### 3.1 품질 vs 사용감/성능

```
품질: 제품이 '정상 상태'가 아니다 (결함/하자/이상)
  - "불량이에요", "변질됐어요", "유통기한 지났어요"
  - "깨짐 없이 잘 왔어요" (긍정)

사용감/성능: 제품은 정상인데, 기대한 '효과/체감'이 좋거나 나쁘다
  - "효과가 없어요", "보습력이 별로예요"
  - "잘 발려요", "지속력이 좋아요"
```

### 3.2 품질 vs 배송/포장

```
배송/포장: 배송 과정에서 발생한 문제
  - "파손되어 왔어요", "뚜껑이 풀려있었어요"
  - "박스가 젖어있었어요", "깨져서 왔어요"

품질: 제품 자체의 결함 (배송과 무관)
  - "변질된 것 같아요", "유통기한이 지났어요"
  - "정품인지 의심스러워요"
```

### 3.3 재구매 태그 vs 재구매 의사

```
재구매 의사 O: 실제로 다시 구매하겠다는 표현
  - "또 살 거예요", "추천합니다", "쟁여둬야겠어요"

재구매 의사 X: 단순히 "재구매"로 시작하는 태그
  - "재구매 아직 안 써봤어요" → 재구매 aspect 제외
  - "재구매 그냥 그래요" → 재구매 aspect 제외
```

## 4. Sentiment Score 기준

### 4.1 점수 범위

| 범위 | 의미 | 예시 표현 |
|------|------|-----------|
| 1.0 | 강한 긍정 | 최고, 대박, 완벽, 인생템, 강추 |
| 0.8 | 일반 긍정 | 좋아요, 만족해요, 괜찮아요 |
| 0.0 | 중립 | 보통이에요 |
| -0.3 | 약한 부정 | 그냥 그래요, 쏘쏘, 별로 |
| -0.7 | 일반 부정 | 실망, 아쉬워요 |
| -1.0 | 강한 부정 | 최악, 짜증, 환불, 다시는 안 삼 |

### 4.2 주의사항

```
"그냥 그래요" = 약한 부정 (-0.3)
"모르겠어요" = 약한 부정 (-0.3)
"보통이에요" = 중립 (0.0)
```

## 5. Aspect별 평가 포인트

### 5.1 사용감/성능

```
✓ 확인: 발림, 지속, 커버, 흡수, 촉촉, 건조, 효과
✗ 주의: 색상 관련은 "색상/발색"으로
✗ 주의: 향/냄새 관련은 "재질/냄새"로
```

### 5.2 색상/발색

```
✓ 확인: 색, 컬러, 톤, 발색, 쿨톤, 웜톤
✓ 확인: 핑크, 레드, 코랄 등 색상명
✗ 주의: 패키지 색상은 "디자인"으로
```

### 5.3 재구매

```
✓ 확인: 또 살, 다시 사, 추천, 계속 쓸, 쟁여
✗ 주의: 단순 "재구매" 태그는 제외
✗ 주의: "안 살 것"도 재구매 Aspect에 포함 (부정)
```

### 5.4 품질

```
✓ 확인: 불량, 변질, 유통기한, 정품, 결함
✗ 주의: "효과 없음"은 "사용감/성능"
✗ 주의: 배송 중 파손은 "배송/포장"
```

### 5.5 배송/포장

```
✓ 확인: 배송, 택배, 포장, 박스, 파손
✓ 확인: 배송 중 파손, 깨짐, 뚜껑 풀림
✗ 주의: 제품 패키징은 "디자인"으로
```

## 6. 오분류 패턴

### 6.1 흔한 오분류

| 오분류 | 올바른 분류 | 예시 |
|--------|-------------|------|
| 품질 → 사용감 | 사용감/성능 | "효과가 없어요" |
| 품질 → 배송 | 배송/포장 | "파손되어 왔어요" |
| 사용감 → 색상 | 색상/발색 | "발색이 예뻐요" |
| 디자인 → 색상 | 색상/발색 | "색이 예뻐요" |
| 사용감 → 재질 | 재질/냄새 | "향이 좋아요" |

### 6.2 경계 케이스

| 케이스 | 판단 기준 |
|--------|-----------|
| "예쁘다" | 제품 색상 → 색상/발색, 패키지 → 디자인 |
| "파손" | 배송 중 → 배송/포장, 제품 자체 → 품질 |
| "냄새" | 제품 향 → 재질/냄새, 변질 → 품질 |
