## 1. 연구 배경

### 1.1 시장 현황

다이소 뷰티 제품 매출은 전년 대비 217% 증가하며 급성장하고 있으며, 초저가 화장품 시장이 주요 소비 트렌드로 부상하고 있습니다. 그러나 신세계, 롯데 등 주류 유통회사들이 저가 화장품 시장에 적극 진출하면서 가격만으로는 경쟁 우위를 유지하기 어려운 상황입니다.

### 1.2 선행 연구 요약

#### 논문 1: 다이소 화장품 선택속성이 재구매의도에 미치는 영향 (2025)

- 품질, 디자인, 편의성은 재구매 의도에 긍정적 영향
- 가격과 브랜드 이미지는 재구매 의도에 유의미한 영향 없음
- 저가 제품이라도 실용적 가치와 사용자 경험이 중요

#### 논문 4: 다이소몰 리뷰 빅데이터 분석 (2025)

- 만족 키워드: '좋다', '꾸준하다', '촉촉하다' 등
- 불만 키워드: '자극', '여드름' (품질), '배송', '재고' (물류)
- 소비자는 고가 제품의 대체품(DUPES)으로 인식하며 품질 기대

#### 논문 5: K-뷰티 저가 화장품 글로벌 브랜드화 전략 (2017)

- 해외 성공 사례: 현지화, 독특한 디자인, 스토리텔링이 핵심
- 단순 한류 편승이 아닌 제품 자체의 독창성 필요

### 1.3 연구의 필요성

선행 연구들은 설문조사 또는 제한된 제품의 리뷰 분석에 기반하고 있습니다. 본 연구는 다이소몰 전체 뷰티 제품의 대규모 리뷰 데이터를 활용하여 다음을 확인합니다:

1. 재구매를 만드는 핵심 속성이 실제 리뷰에서 어떻게 나타나는가?
2. 다이소 뷰티가 저가 소비 시장인가, 듀프(대체재) 시장인가?

## 2. 연구 질문

### RQ1: 재구매를 만드는 핵심 속성은 무엇인가?

**배경:**

- 다이소의 핵심 경쟁력은 가격이 아니어야 함
- '싸서 한번 사는 시장'이 아니라 '품질이 괜찮아서 계속 쓰는 시장'으로 전환 필요

**세부 질문:**

- 재구매 그룹에서 '가성비' 키워드보다 '품질', '심미' 키워드가 더 많이 등장하는가?
- 물류 불만과 제품 품질 불만을 분리할 수 있는가?
- "제품은 좋은데 구하기 힘들다(희소성)" 패턴이 존재하는가?

### RQ2: 저가 소비 시장인가, 듀프 소비 시장인가?

**배경:**

- 화장품 회사 협업이 단순 저가 생산이 아닌 독특한 컨셉 실험 무대일 수 있음
- 브랜드 이미지보다 제품력이 중요하다면 중소 브랜드에게 기회

**세부 질문:**

- 다이소 제품이 어떤 고가 브랜드의 대체재로 언급되는가?
- 올리브영 vs 다이소 이동 경로가 나타나는가?
- 브랜드 유무가 만족도에 영향을 미치는가?

- 정규표현식 기반 키워드 추출
- 맥락 분석 (전후 텍스트 확인)
- 동시 출현 패턴 분석

## 6. 분석 설계

### 6.1 RQ1 분석 설계

**목적**: 재구매를 만드는 핵심 속성 파악

**분석 단계**:

1. 재구매 여부 식별
   - 리뷰 맨 앞 "재구매" 키워드 감지
   - is\_재구매 플래그 생성

2. 그룹 분리 및 기본 통계
   - 재구매 그룹 vs 일반 그룹
   - 평점, 리뷰 길이, 이미지 포함 비율 비교

3. 키워드 빈도 분석
   - 그룹별 상위 키워드 50개 추출
   - 그룹 간 키워드 빈도 차이 계산
   - 워드클라우드 시각화

4. 카테고리별 키워드 분석
   - 가성비 vs 품질/심미/편의 키워드 비율
   - 재구매 그룹에서 강조되는 카테고리 파악

5. 물류 vs 품질 불만 분리
   - 물류 키워드와 품질불만 키워드 별도 추출
   - 평점별 물류/품질 불만 비율 분석
   - 희소성 패턴 (물류 불만 + 높은 평점) 확인

**예상 결과**:

- 재구매 그룹에서 품질/심미/편의 키워드 비율이 더 높을 것
- 물류 불만은 제품 품질과 독립적일 것
- 높은 평점 + 물류 불만 = 희소성 패턴

### 6.2 RQ2 분석 설계

**목적**: 다이소 뷰티 시장 특성 파악 (저가 vs 듀프)

**분석 단계**:

1. 고가 브랜드 언급 패턴
   - 20개 고가 브랜드 키워드 정의
   - 리뷰에서 브랜드 언급 빈도 계산
   - 어떤 제품이 어떤 브랜드의 대체재로 언급되는지 파악

2. 올리브영 언급 분석
   - '올영', '올리브영' 키워드 추출
   - 비교 맥락 분석 ('올영보다', '올영 대신' 등)
   - 경쟁자 vs 보완재 인식 파악

3. 듀프 키워드 분석
   - 듀프 관련 키워드 7개 정의
   - 듀프 언급 리뷰 추출 및 제품 순위
   - 듀프 언급 시 평점 분포

4. 브랜드 vs 제품력 분석
   - 브랜드 키워드 vs 품질 키워드 언급 비율
   - 브랜드 유무별 평점 비교
   - 주요 브랜드별 평점 분석

**예상 결과**:

- 듀프 시장 특성이 나타날 것
- 브랜드보다 제품력 키워드가 더 많이 언급될 것
- 브랜드 유무는 평점에 영향을 미치지 않을 것

### 8.1 RQ1 예상 인사이트

**핵심 발견**:

- 재구매 그룹은 '가성비'보다 '품질', '디자인', '편의성' 키워드를 더 많이 언급
- 물류 불만과 제품 품질 불만은 분리됨 (물류는 만족도와 독립적)
- "제품은 좋은데 구하기 힘들다"는 희소성 패턴 존재

**비즈니스 시사점**:

1. 가격 경쟁력만으로는 재구매 유도 불가
2. 품질, 디자인, 편의성 개선에 집중 필요
3. 물류 개선은 고객 만족도 향상에 기여하나 재구매 직접 요인은 아님
4. 인기 제품의 재고 확보가 중요 (희소성이 긍정적 효과)

### 8.2 RQ2 예상 인사이트

**핵심 발견**:

- 소비자는 다이소 제품을 고가 브랜드의 대체재(듀프)로 인식
- 브랜드 언급보다 제품력 언급이 훨씬 많음
- 브랜드 유무는 평점에 유의미한 영향 없음
- 올리브영은 주로 비교 대상 (경쟁자)으로 언급

**비즈니스 시사점**:

1. 듀프 마케팅 강화: "XX 브랜드 대체재" 포지셔닝
2. 중소 브랜드 파트너십 기회: 유명 브랜드 아니어도 품질로 승부 가능
3. 제품력 중심 홍보: 브랜드보다 효과, 품질 강조
4. "올리브영 제품을 다이소 가격에" 마케팅 활용

### 8.3 종합 시사점

**다이소 뷰티 시장의 특성**:

- "브랜드 충성" 시장이 아닌 "제품 단위 충성" 시장
- 가격은 기본 조건, 품질이 차별화 요인
- 중소 뷰티 브랜드에게 매력적인 진입 채널

**전략 방향**:

1. 품질 관리 강화
2. 독특한 컨셉 제품 개발 (비건, 여행용 등)
3. 디자인 및 패키징 개선
4. 물류 시스템 최적화
5. 중소 브랜드 발굴 및 협업 확대

## 9. 한계점 및 후속 연구

### 9.1 연구의 한계

1. **데이터 한계**
   - 온라인 리뷰만 분석 (오프라인 구매 의견 미반영)
   - 자발적 리뷰 작성자 편향 (극단적 의견 과대표현 가능)
   - 크롤링 시점 제한 (최신 데이터만 포함)

2. **방법론 한계**
   - 탐색적 분석으로 인과관계 입증 어려움
   - 키워드 사전이 연구자 주관에 의존
   - 형태소 분석기의 오류 가능성

3. **맥락 이해 한계**
   - 단순 키워드 빈도로는 문맥 파악 제한적
   - 부정문 처리 어려움 ("좋지 않다"를 "좋다"로 인식 가능)

### 9.2 후속 연구 제안

1. **감성 분석 추가**
   - BERT 기반 감성 분류 모델 적용
   - 긍정/부정 맥락 정확히 분리

2. **토픽 모델링**
   - LDA를 활용한 잠재 주제 추출
   - 제품 카테고리별 주요 토픽 비교

3. **시계열 분석**
   - 재구매율 트렌드 변화
   - 듀프 인식 증가 추이

4. **제품별 상세 분석**
   - 베스트셀러 제품의 리뷰 패턴
   - 카테고리별 차별화 전략

5. **사용자 세분화**
   - RFM 분석 기반 고객 군집화
   - 재구매 고객의 구매 패턴 분석
