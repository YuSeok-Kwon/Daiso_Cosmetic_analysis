## 프로젝트 개요

**"모든 유기동물에게 최적의 가족을, 모든 예비 입양자에게 최적의 반려동물을"**

1. 어떤 특성을 가진 유기동물이 빨리 입양되는가?

## 머신러닝 모델 상세

### 입양 성공 예측 (정형 데이터)

입력 변수
├── 동물 특성: 품종, 나이, 성별, 중성화, 체중, 색상
├── 보호 정보: 보호 기간, 보호소 지역
└── 시간 정보: 공고 계절, 요일

출력: 입양 확률 (0~100%)

모델: XGBoost, Random Forest / ResNet/EfficientNet (Transfer Learning) / KoBERT 감성 분석
평가: AUC-ROC 0.85+, Precision-Recall

| **구분**        | **상세 내용**                                                                                                                                                                  | **평가/수치**                      |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------- |
| **기본 정보**   | **공고번호** `서울-종로-2024-00123` 말티즈 / 2세 / 암컷 / 중성화(O)                                                                                                            | 보호기간: 5일차                    |
| **사진 분석**   | • 조명: 양호 (밝음) • 배경: 깔끔함•**[개선점]** 정면 촬영이 아님                                                                                                               | **⭐⭐⭐⭐☆ (4.0)** _(우수함)_     |
| **텍스트 분석** | • 긍정 키워드: "온순한", "사람 좋아함" • 부정 뉘앙스: "예민한", "적응 필요" •**[개선점]** 성격 묘사 부족                                                                       | **⭐⭐⭐☆☆ (3.0)** _(보통)_        |
| **예측 결과**   | **현재 예상 입양 확률**  **52%** (유사품종 평균 대비 낮음)                                                                                                                     | **개선 후 예상** : **71% (+19%p)** |
| **AI 제안**     | **[문구 자동 수정 제안]** _기존:_ "좀 예민하고 낯선 환경에 적응이 필요합니다" *제안:* **"새로운 가족에게 천천히 마음을 여는 신중한 성격입니다. 조용한 환경에서 더욱 빛나요!"** | **즉시 적용**                      |

### 사용자-동물 매칭 추천

알고리즘: Content-Based + Collaborative Filtering 하이브리드

입력
├── 사용자 프로필 (심리테스트 결과)
│ ├── 주거환경 벡터
│ ├── 생활패턴 벡터
│ ├── 성격 선호 벡터
│ └── 경험 수준
│
└── 동물 프로필
├── 물리적 특성 벡터
├── 성격 태그 벡터
└── 돌봄 난이도

출력: 매칭 점수 상위 5마리 + 매칭 이유
